Django Roman
************

**Django Roman** is a `Django <https://www.djangoproject.com/>`_ `template tag <https://docs.djangoproject.com/en/dev/howto/custom-template-tags/>`_ application for converting `Arabic numerals <https://en.wikipedia.org/wiki/Arabic_numerals>`_ to `Roman numerals <https://en.wikipedia.org/wiki/Roman_numerals>`_.
